# 🍽️ PLATE

고등학생 수준에서 이해하고 유지보수하기 쉽게 설계된, 그러나 시각적으로는 **가장 트렌디하고 유쾌한** 맛집 추천 웹사이트입니다.

## ✨ 주요 특징

### 1. 스켈레톤 로딩 (Skeleton UI)
- 외부 카카오맵 API에서 맛집 데이터를 검색하고 불러오는 동안, 빈 화면 대신 **스켈레톤 로딩 UI**를 제공합니다.
- 사용자의 테마에 맞추어 다크 모드 전용 스켈레톤 색상까지 자연스럽게 지원합니다.

### 2. 사용자에게 웃음을 주는 유쾌한 마이크로 인터랙션 (Micro-interactions)
단순하고 밋밋한 웹사이트에서 벗어나, 마우스를 올리거나 버튼을 누를 때마다 소소한 재미를 주는 기발한 애니메이션 효과들을 듬뿍 담았습니다.
- **메가 붐(Mega Boom) 잭팟**: 랜덤 추천 버튼을 누르면 카드가 하늘에서 거대한 크기로 나타나 바닥에 **쾅!** 하고 내리꽂히며 요동칩니다.
- **안달 난 버튼**: '랜덤으로 추천받기' 버튼에 마우스를 올리면 당장 눌러달라는 듯 격렬하게 바들바들 떱니다.
- **로고 턴 & 지역 아이콘 점프**: 로고에 마우스를 올리면 아이콘이 펄쩍 뛰며 한 바퀴 돌고, 지역 버튼을 호버하면 아이콘들이 신나게 방방 뜁니다.
- **두근거리는 찜(하트)**: 맛집을 찜하면 빨간 하트가 실제 심장처럼 두근두근 박동합니다.

### 3. 직관적이고 분리된 파일 구조
고등학교 1~2학년 학생들도 쉽게 코드를 읽고 수정할 수 있도록 복잡한 구조를 최대한 덜어내고 직관적으로 재편했습니다.
- **`data.js`의 분리**: 맛집의 모든 정보(이름, 주소, 메뉴, 사진 등)가 모여있는 파일입니다. 복잡한 JS/CSS 로직을 건드릴 필요 없이 이 데이터 파일만 수정하면 새로운 맛집을 손쉽게 추가할 수 있습니다.
- HTML 파일에서 지저분한 인라인 스타일(`style="..."`)을 모두 제거하고, 각 페이지별로 독립된 깔끔한 CSS 및 JS 파일을 갖도록 분리했습니다.

### 4. 핵심 기능
- **오늘 뭐 먹지? (지역별 랜덤 추천)**: 원하는 지역을 선택하고 버튼을 누르면 해당 지역의 맛집을 무작위로 하나 꼽아줍니다.
- **나만의 찜 목록 보관**: 마음에 드는 맛집을 찜하면 브라우저의 `localStorage`를 통해 저장되어 브라우저를 껐다 켜도 언제든 다시 볼 수 있습니다.
- **카카오맵 연동 상세 페이지**: 맛집 상세 페이지에서 카카오맵 API를 통해 정확한 위치를 확인할 수 있고, 동적인 탭 UI를 통해 메뉴와 리뷰를 살펴볼 수 있습니다.
- **검색 및 필터링 기능**: 맛집 찾기 페이지에서 검색어와 지역/카테고리 필터를 통해 원하는 식당을 빠르게 탐색합니다.

## 📁 프로젝트 파일 구조

```text
TeamPlay/
│
├── index.html       # 메인 홈 화면 (슬라이드 배너, 랜덤 추천, 요즘 뜨는 맛집)
├── list.html        # 맛집 찾기 화면 (검색, 필터, 전체 목록 그리드)
├── detail.html      # 맛집 상세 화면 (카카오맵, 상세 메뉴, 리뷰)
├── bookmark.html    # 찜 목록 화면 (찜한 맛집 모아보기)
│
├── css/
│   ├── common.css   # 전체 공통 디자인 (글로벌 변수, 다크모드, 헤더/푸터, 공통 애니메이션)
│   ├── index.css    # 메인 페이지 전용 디자인 (메가 붐 효과 등)
│   ├── list.css     # 맛집 찾기 페이지 레이아웃 및 검색바 스타일
│   ├── detail.css   # 상세 페이지 탭, 리뷰 폼, 별점 스타일
│   └── bookmark.css # 찜 목록 전용 디자인 (데이터 없을 때의 빈 화면 처리 등)
│
└── js/
    ├── data.js      # 💡 식당 데이터베이스 (데이터 추가/수정은 여기서!)
    ├── common.js    # 공통 기능 (다크모드 토글 등)
    ├── index.js     # 메인 페이지 로직 (랜덤 추천 필터링 및 선택 로직)
    ├── list.js      # 맛집 목록 렌더링 및 다중 필터/검색 기능
    ├── detail.js    # 맛집 상세 정보 표시, 찜하기 기능, 카카오맵 로드
    └── bookmark.js  # 로컬 스토리지의 찜한 식당 목록 불러오기
```
