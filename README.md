# PLATE. - 프리미엄 다이닝 탐색 플랫폼

> **고등학교 팀 프로젝트 (프론트엔드 웹 개발)**
> 당신만의 특별한 공간을 찾아주는 프리미엄 다이닝 큐레이션 서비스입니다.

## 팀원 소개
- **김주완 (고2)** - 개발 리드 (프론트엔드 기획 및 핵심 로직 구현, API 연동)
- **팀원 (고1)** - 디자인 보조 (UI/UX 디자인 기획, 아이콘 및 컬러 시스템 보조)

## 기술 스택 (Tech Stack)
- **Core:** HTML5, CSS3, Vanilla JavaScript
- **API:** Kakao Local API (키워드 장소 검색), Kakao Maps API (지도 렌더링)
- **State Management:** 브라우저 Local Storage
- **Design:** CSS Variables, Glassmorphism, SVG Icons

## 핵심 기능 및 구현 기술 (Key Features)

### 1. 완벽한 다크 모드 (Dark Mode)
- 브라우저의 `Local Storage`를 활용하여 사용자가 선택한 테마(라이트/다크) 상태를 저장하고 웹사이트 재방문 시에도 유지되도록 구현했습니다.
- CSS 변수(`--bg-main`, `--text-main` 등)를 활용하여 모드 전환 시 부드럽고 일관된 컬러 시스템이 적용되게 만들었습니다.

### 2. 카카오 API 실시간 검색 및 지도 연동
- **Kakao Local API**를 비동기(`async/await`, `fetch`)로 호출하여 실시간으로 전국 맛집을 검색합니다.
- 상세 페이지에서는 **Kakao Maps API**를 통해 선택한 다이닝의 실제 위치를 마커와 함께 렌더링합니다.

### 3. 스켈레톤 로딩 UI (Skeleton UI)
- API 통신 중 발생하는 지연 시간 동안 빈 화면이나 구식 스피너 대신, 카드 뼈대 형태의 **스켈레톤 로딩 애니메이션**을 보여주어 체감 대기 시간을 줄이고 사용자 경험(UX)을 대폭 개선했습니다.

### 4. 나만의 장소 저장 (Bookmark)
- 마음에 드는 다이닝 카드의 하트를 누르면 `Local Storage` 배열에 데이터가 영구 저장됩니다.
- [찜] 탭에서 내가 찜한 목록만 필터링하여 갤러리 형태로 모아볼 수 있고, 실시간으로 찜 해제가 연동됩니다.

### 5. 스크롤 리빌 애니메이션 (Scroll Reveal)
- 순수 자바스크립트의 `Intersection Observer API`를 사용하여 사용자가 화면을 스크롤할 때 요소가 화면 뷰포트에 나타나면 아래에서 위로 부드럽게 페이드업(Fade-up) 되는 고급스러운 인터랙션을 적용했습니다.

### 6. 반응형 웹 디자인 (Responsive Web)
- Media Query(`@media`)와 Flex/Grid 레이아웃을 적극 활용하여 PC, 태블릿, 스마트폰 등 다양한 디바이스 환경에서 UI가 최적화되어 보이도록 구현했습니다.

## 프로젝트 기획 의도
흔한 '맛집 리스트' 앱 디자인에서 벗어나, 유명 파인다이닝 사이트나 고급 매거진 같은 **하이엔드 감성의 프리미엄 웹 플랫폼**을 목표로 했습니다. 

무겁고 화려한 프레임워크(React, Vue 등)에 의존하지 않고, 순수 **바닐라 자바스크립트(Vanilla JS) -> 오직 자바스크립트로만,,**와 CSS만을 이용하여 비동기 통신, 상태 관리, 옵저버 패턴 등 프론트엔드 웹 개발의 핵심 기술들을 직접 구현하며 프로그래밍 기본기를 깊이 있게 다지는 데 집중했습니다.
